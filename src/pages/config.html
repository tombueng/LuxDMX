Display
Shows IP, universe, FPS, source count and link status; auto-switches to a warning banner on a source conflict, identify, or manual override. I²C address is auto-detected (0x3C / 0x3D); the colour panel mirrors the status-LED colours.

DMX Outputs
Each output is an independent DMX universe driven by its own ESP32 UART + RS485 transceiver. Up to 2 outputs are supported (the chip has 3 UARTs and one is the serial console). Pin numbers are ESP32 GPIO numbers (the GPIO xx / Gxx label printed on the board), matched to the transceiver pins by function: TX → DI (driver input), RX → RO (receiver output, needed for RDM), RTS → EN (direction / DE-RE; set it to enable RDM, leave at −1 for auto-direction modules such as the Waveshare C). Each enabled output needs a distinct UART port. RDM runs on the first enabled output that has an RTS pin set. Wire the RS485 A/B → XLR pin 3/2, GND → XLR pin 1, and add a 120 Ω terminator at the end of each chain.
