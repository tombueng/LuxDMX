# Board photo credits

The pin picker's board diagrams are our own generated SVG (no third-party assets).
The optional, online-only board **photos** come from these sources:

| File | Board | Source | License |
|---|---|---|---|
| `img/lumigate-v3.png` | LumiGate v3 | our own KiCad render (`hardware/board-pcb-1.png`) | project (MIT, same as repo) |
| `img/esp32-devkitc.jpg` | ESP32 DevKitC (WROOM-32) | [Wikimedia Commons](https://commons.wikimedia.org/wiki/File:ESP32_Espressif_ESP-WROOM-32_Dev_Board.jpg) by Ubahnverleih | CC0 1.0 (public domain) |

Notes:
- `esp32-devkitc.jpg` is downscaled to 800 px from the CC0 original; CC0 requires no
  attribution, this table is courtesy/transparency.
- Photos are served from GitHub Pages and loaded only when the user opts in; they are
  never embedded in the firmware flash.
- Only add photos from sources we may legally redistribute (own renders, CC0, or
  CC-BY/CC-BY-SA **with** attribution shown). Otherwise leave a board photo-less; the
  SVG diagram is the actual interactive tool.
