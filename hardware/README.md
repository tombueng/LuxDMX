# LumiGate carrier PCB — design source

Schematic-as-code for the Olimex ESP32-POE-ISO carrier board (isolated ADM2587E
DMX/RDM out, USB-C power, WS2812 RGB). Full design rationale: [../docs/hardware.md](../docs/hardware.md).

## Files
| File | What |
|---|---|
| `lumigate_carrier.py` | SKiDL source — the authoritative netlist definition |
| `lumigate_carrier.net` | Generated KiCad netlist (import into Pcbnew to start layout) |
| `build_pcb.py` | Builds a populated, pre-placed `lumigate_carrier.kicad_pcb` from the netlist via KiCad's `pcbnew` Python API (run with the KiCad 10 bundled python). Footprints loaded, nets assigned, parts grouped, board outline drawn — ready to route. |
| `lumigate_carrier.kicad_pcb` | The placed board (generated by `build_pcb.py`). Open in KiCad to route. |
| `make_bom.py` → `lumigate_carrier_BOM_jlcpcb.csv` | JLCPCB assembly BOM (SMT parts grouped; THT/hand-solder parts listed separately). |
| `lumigate_carrier_CPL_jlcpcb.csv` | Component placement (CPL) exported from the board via `kicad-cli pcb export pos`. |
| `make_schematic.py` → `lumigate_carrier_schematic.kicad_sch` / `.pdf` | Flat schematic (each part a labelled box, every pin tagged with its net = connectivity-by-name). Generated from the netlist; opens in Eeschema, exported to PDF. |
| `gerbers/` + `lumigate_carrier_gerbers.zip` | Gerbers + drill (`kicad-cli pcb export`), zipped for JLCPCB/PCBWay upload. |
| `route.py` | Headless auto-route of an existing board (DSN → Freerouting → SES → fill). |
| `optimize.py` | **Full auto: place + autoroute + shrink loop.** Tries board sizes smallest-first, places all parts (connectors on edges, openings out), autoroutes, refills zones, runs DRC (rejects unconnected/clearance/shorts/overlaps), keeps the smallest size that passes. Result: **108 × 70 mm**, fully routed, only cosmetic silk/thermal/edge DRC items left. Needs KiCad 10 python + java + freerouting 1.9.0 jar (`FREEROUTING_JAR`). |

`.erc` / `.log` / `*_sklib.py` are SKiDL byproducts (git-ignored).

## Regenerate the netlist
```bash
python -m venv venv && venv/Scripts/pip install skidl
venv/Scripts/python lumigate_carrier.py     # writes lumigate_carrier.net
```
ERC result: **0 errors**. The remaining warnings are expected — "insufficient drive"
on the isolated grounds / 5 V rail (no `PWR_FLAG` in a bare netlist; harmless) and the
intentionally-open WS2812 `DOUT`.

### U1 socket footprint pad map (pads 1–10 = EXT1, 11–20 = EXT2)
```
EXT1: 1 +5V   2 +3V3  3 GND   4 ESP_EN 5 GPIO0 6 GPIO1 7 GPIO2 8 GPIO3 9 GPIO4  10 GPIO5
EXT2: 11 GPIO39 12 GPIO36 13 GPIO35 14 GPIO34 15 GPIO33 16 GPIO32 17 GPIO16 18 GPIO15 19 GPIO14 20 GPIO13
```
Used: pad1=+5V, pad2=+3V3, pad3=GND, pad9=GPIO4(DI), pad12=GPIO36(RO), pad15=GPIO33(RGB), pad16=GPIO32(DE/RE).

## Pinout provenance
U2 (ADM2587E) pin numbers are taken from the ADI ADM2582E/ADM2587E datasheet
pin-configuration table (20-lead SOIC_W): VISOOUT=12 / VISOIN=19 (tied), Y=13/A=18 (Data+),
Z=15/B=17 (Data−), GND1=1/3/9/10, GND2=11/14/16/20, VDD=2/8, RxD=4, /RE=5, DE=6, TxD=7.
